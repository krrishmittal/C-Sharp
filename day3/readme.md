# Day 3 - Arrays, Methods, and Classes

## Topics Covered

### 1. **String Arrays**
   - Creating and working with arrays of strings
   - Taking user input for string arrays
   - Storing text data in collections

### 2. **2D Arrays**
   - Working with two-dimensional arrays using nested loops
   - Representing data in matrix format
   - Accessing elements using row and column indices

### 3. **Jump Statements**
   - Using `break` to exit loops
   - Using `continue` to skip iterations
   - Using `goto` to transfer control to labeled statements
   - Controlling program flow with jump statements

### 4. **Array Sorting**
   - Using `Array.Sort()` to sort arrays in ascending order
   - Organizing array data
   - Preparing arrays for binary search operations

### 5. **Binary Search**
   - Using `Array.BinarySearch()` to search for elements in sorted arrays
   - Efficient searching in large datasets
   - Returning index of found elements

### 6. **Classes and Objects**
   - Understanding how classes define blueprints
   - Objects as instances with allocated memory
   - Class vs instance concepts
   - Creating and using objects

### 7. **Class Members**
   - Working with fields, properties, and methods within classes
   - Encapsulating data and behavior
   - Access modifiers for class members

### 8. **Methods in C#**
   - Creating reusable blocks of code with parameters and return types
   - Method signatures and declarations
   - Passing data to methods and receiving results

### 9. **Method Calling**
   - **Direct Method Calling** - Invoking instance methods using object references
   - **Static Method Calling** - Invoking static methods using class names without instantiation

### 10. **Method Parameters**
   - **Value Parameters** - Pass copies of arguments
   - **Reference Parameters** - Pass references using `ref` keyword
   - **Output Parameters** - Return multiple values using `out` keyword
