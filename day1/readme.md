# Day 1 - C# Basics

## Topics Covered

### 1. **Printing Statements**
   - Using `Console.WriteLine()` to output text to the console
   - Displaying messages and data to users
   - Formatting output for readability

### 2. **User Input**
   - Taking input from users with `Console.ReadLine()`
   - Storing and displaying user input
   - Interactive console applications

### 3. **Integer Calculations**
   - Creating custom classes and methods for arithmetic operations
   - Addition, subtraction, multiplication, and division
   - Building reusable calculation logic
   - Method parameters and return values

### 4. **Control Statements**
   - Using `if-else` conditions for decision-making logic
   - Making programs respond to different inputs
   - Nested conditional statements

### 5. **Loops**
   - Implementing `for` loops for iteration
   - Repeating code blocks multiple times
   - Controlling loop flow with conditions

### 6. **Classes in C#**
   - Defining and using classes with methods
   - Organizing code into reusable units
   - Object-oriented programming fundamentals
   - Methods as functions within classes

### 7. **Variables**
   - Working with different data types (int, string, bool, double)
   - Declaring and initializing variables
   - Type safety in C#
   - Variable naming conventions

### 8. **Arrays**
   - Creating and iterating through arrays using loops
   - Storing multiple values of the same type
   - Accessing array elements using indices
   - Array iteration with for loops
