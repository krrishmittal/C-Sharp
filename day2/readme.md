# Day 2 - C# Intermediate Concepts

## Topics Covered

### 1. **Classes and Objects**
   - Creating classes and instantiating objects in C#
   - Defining blueprints for objects
   - Creating multiple instances from a single class definition

### 2. **Inheritance**
   - Using class inheritance to extend functionality from parent classes
   - Reusing code from existing classes
   - Creating class hierarchies
   - Base class and derived class relationships

### 3. **If-Else Statements**
   - Conditional logic for decision-making
   - Multiple conditions and nested statements
   - Making programs behave differently based on conditions

### 4. **Out Parameters**
   - Using `out` keyword to return multiple values from a method
   - Passing parameters by reference for output
   - Returning data without using traditional return statements

### 5. **Arrays**
   - Creating and working with arrays
   - Taking array input from users
   - Storing collections of data
   - Efficient data management

### 6. **Foreach Loops**
   - Iterating through array elements using `foreach` loop
   - Simplified iteration syntax compared to traditional for loops
   - Working with collections without index management

### 7. **Params Keyword**
   - Using `params` to accept a variable number of arguments in a method
   - Creating flexible method signatures
   - Passing different numbers of parameters to a single method

### 8. **Method Return Values**
   - Returning and handling multiple values from methods
   - Understanding return types
   - Using out and ref keywords for multiple returns
